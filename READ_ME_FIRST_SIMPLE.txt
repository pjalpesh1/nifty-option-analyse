NIFTY OPTION AUTO UPLOAD - SIMPLE USE

1. Double-click START_HERE_AUTO_UPLOAD.bat
2. Excel will open.
3. Go to sheet OptionData.
4. Paste or update option-chain rows.
5. Save Excel.
6. Keep the black uploader window open.
7. Open https://pjalpesh1.github.io/nifty-option-analyse/
8. In Data tab keep URL: data/nifty-options.dat
9. Click Load URL, then Auto Start.

Important:
- If Excel has 0 data rows, website will show no data.
- If you want to test, copy rows from sheet Sample into sheet OptionData and save Excel.
- Save Excel after every update.
- GitHub login is needed only one time.
- Local uploaded files cannot auto-update. The website reads the hosted DAT file.
